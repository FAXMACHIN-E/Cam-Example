Image based ASL Interpretor Web Application


  • Goals

    • Ideal Use Case (Long Term)

    To build an image recognization Web App to help deaf & mute people communicate, using ASL, with those who doesn't know ASL. The App will accept input from both live-stream and video upload.

    Why is this useful? While we can ask deaf & mute people to type, it'd be greatly useful if they don't touch keyboards in the middle of a presentation going through slides or annotating on a whiteboard.

    MVP (Now)

    To build an image recognization Web App to predict the predicted English based on the uploaded static IMAGE containing relevant ASL signs.

    Final Project Plan (by the End of the Summer)

    To build an image recognization Web App to predict the predicted English based on the VIDEO containing relevant ASL signs. The App will accept input from live-stream, video upload or both.



  • Illustration of A Similar Working Example

    • We'll provide web interface for users to upload files or connect with his/her PC's camera. For each of the frames in the video, we interpret the American Sign Language gesture into an English letter (or words and phrases in the future).
